## 📦 安装方法

### 方式一：下载即用（推荐）

1. 点击页面右上角 **Code → Download ZIP**
2. 解压到任意文件夹
3. 双击 `generate_5w.py` 即可运行（需安装 Python）

### 方式二：Git克隆

```bash
git clone https://github.com/Jack-the-slp/5W-Q-A.git
cd 5W-Q-A
```

### 环境要求

- Python 3.7+
- python-docx 库

```bash
pip install python-docx
```

## 📖 使用方法

### 快速开始

```bash
# 进入目录
cd 5W-Q-A

# 直接运行（生成全部9个场景的Word图卡到桌面）
python generate_5w.py
```

生成的Word文件会自动保存到 **桌面**，文件名格式：`5W_场景名.docx`

### 自定义场景

编辑 `5w_scenes.json` 文件，可以：

- **修改问题内容**：根据孩子能力调整问题难度
- **添加新场景**：参考现有格式添加新场景
- **调整语言风格**：简化或丰富问题表述

### 打印使用

1. 用 Word 打开生成的 `.docx` 文件
2. 直接打印（建议A4纸）
3. 训练时引导孩子边看图边回答问题，用笔勾选已回答的问题

## 🔬 训练理论依据

5W提问法基于以下言语康复理论：

- **儿童语言发展规律**：从指认（What）→ 命名（Who）→ 空间（Where）→ 时序（When）→ 因果（Why），符合语言发育阶梯
- **图片交换沟通系统（PECS）**：以图片为媒介促进沟通动机
- **结构化语言训练**：通过5W框架建立完整句式表达能力
- **游戏化康复**：在趣味场景中减少抗拒，提升训练效果

## 📊 使用建议

| 孩子语言水平 | 推荐训练方式 |
|-------------|------------|
| **刚开口** | 从"谁""什么"开始，简单指认 |
| **有基础表达** | 增加"哪里""做什么"，引导完整句子 |
| **表达较好** | 加入"为什么"，培养逻辑思维 |
| **泛化阶段** | 带孩子到真实超市、公园，用同样框架观察真实场景 |
